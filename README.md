
# DTL Analysis Tool

## 概要
このプログラムは、デュエル・マスターズ（DTL）というゲームにおける **先手後手の影響**、**運の要素**、**プレイヤーの技量**を分析するためのツールです。  
試合結果データを用いて、**ロジスティック回帰モデル**を構築し、勝敗に影響する要因を定量的に評価します。

---

## 主な機能
1. **試合結果の読み込み**  
   - `DTL_match_result.tsv` を読み込み、DataFrameを作成。
   - 先手勝利フラグを生成。

2. **先手勝率の計算**  
   - 全試合における先手勝率を算出。

3. **プレイヤーのレーティング計算**  
   - イロレーティングに基づき、各プレイヤーのレートを更新。

4. **ロジスティック回帰用データ変換**  
   - 試合結果をプレイヤー単位に変換。
   - レーティングを標準化して結合。

5. **ロジスティック回帰モデル構築**  
   - 説明変数：じゃんけん結果、レーティング
   - 目的変数：試合勝敗
   - モデルの係数と寄与率を出力。
